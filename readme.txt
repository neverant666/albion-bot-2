# Bot de Albion Online - Dungeon Auto-Runner

Este bot está diseñado para automatizar la exploración de mazmorras en Albion Online. Una vez que el bot se active dentro de una mazmorra, tomará el control del personaje y realizará las siguientes acciones de forma automática:

1. Navegación automática: El bot utilizará algoritmos de navegación inteligente para moverse a través de la mazmorra, evitando obstáculos y enemigos.

2. Combate automático: Enfrentará a los enemigos dentro de la mazmorra utilizando habilidades predefinidas y tácticas de combate avanzadas.

3. Recolección automática de botín: Recolectará automáticamente cualquier botín o recurso valioso encontrado en la mazmorra.

4. Gestión de recursos: El bot administrará de manera eficiente los recursos del personaje, como salud, energía y habilidades, para asegurarse de que se utilicen de la mejor manera posible durante la exploración de la mazmorra.

5. Finalización de la mazmorra: Una vez que se haya explorado completamente la mazmorra y se hayan cumplido los objetivos, el bot finalizará la instancia y saldrá automáticamente.

## Requisitos del sistema

- Sistema operativo: Windows 10
- Versión de Albion Online: [Versión del juego compatible]
- Conexión estable a Internet

## Instrucciones de uso

1. Descarga el archivo bot-albion.zip adjunto a este readme.

2. Descomprime el archivo en una ubicación de tu elección.

3. Abre el cliente de Albion Online y asegúrate de que tu personaje esté dentro de una mazmorra.

4. Ejecuta el archivo "bot-albion.exe" incluido en la carpeta descomprimida.

5. El bot se activará automáticamente y comenzará a explorar la mazmorra de forma autónoma.

¡Y eso es todo! Ahora puedes sentarte y relajarte mientras el bot hace todo el trabajo duro por ti en Albion Online. Recuerda, el uso de este bot puede violar los términos de servicio del juego, así que úsalo bajo tu propio riesgo y responsabilidad.
